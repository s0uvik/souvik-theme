# Souvik Theme

A custom dark theme for Visual Studio Code with carefully selected background colors and syntax highlighting.

## Installation

1. Copy the `souvik-theme` folder to your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
   - **macOS/Linux**: `~/.vscode/extensions/`

2. Reload VS Code:
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
   - Type "Reload Window" and press Enter

3. Activate the theme:
   - Press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on Mac)
   - Select "Souvik Theme" from the list

## Features

- Custom background colors for optimal readability
- Carefully chosen syntax colors for code clarity
- Dark theme optimized for long coding sessions

## Customization

If you want to modify the theme, edit the `themes/souvik-theme.json` file and reload VS Code.
